## 문제 1. 배열 초기화

다음 코드의 출력 결과를 쓰시오.

```c
#include <stdio.h>

int main() {
    int arr[5] = {1, 2};

    for (int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }

    return 0;
}
```
# 답: 1,2,0,0,0

## 문제 5. 배열과 포인터

다음 코드의 출력 결과를 쓰시오.

```c
#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *p = arr;

    printf("%d\n", *(p + 1));

    return 0;
}
```

# 답: 20
